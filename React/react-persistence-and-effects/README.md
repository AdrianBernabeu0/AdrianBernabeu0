# Persistencia y Efectos (Día 3)
Aplicación de gestión de eventos con almacenamiento persistente, asegurando que los datos no se pierdan al recargar la página.

- **Hook `useEffect`**: Implementación de efectos secundarios para sincronizar el estado con el navegador.

- **LocalStorage**: Uso de la API de almacenamiento web para persistir la lista de eventos en formato JSON.

- **Lazy State Initialization**: Carga optimizada de datos iniciales desde el almacenamiento local al arrancar la aplicación.

- **Sincronización de Título**: Manipulación del DOM (document.title) para reflejar cambios en el estado en tiempo real.

# Instalacion 
- npm install
- npm run dev
