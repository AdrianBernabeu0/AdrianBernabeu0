# Conexión con API a Docker
Migración de datos locales a un servidor externo simulado con Docker.

- **Consumo Asíncrono**: Uso de `fetch` dentro de `useEffect` para las peticiones.
- **Renderizado Dinámico**: Transformación de datos de la API en elementos del DOM mediante `.map()`.

# Instalacion 

### 1. Levantar Servidor (Docker)
- `docker-compose up -d`

### 2. Iniciar Aplicacion
- `npm install`
- `npm run dev`
