# Lógica de Interfaz Dinámica
Aplicación interactiva creada con React y Vite que gestiona el estado de una lista, permitiendo añadir y eliminar eventos en tiempo real.

- **Gestión de Estado (`useState`)**: Sincronización automática entre los datos y la interfaz de usuario.

- **CRUD Básico**: Implementación de lógica para añadir nuevos elementos y filtrar la lista para eliminarlos.

- **Formularios Controlados**: Vinculación de inputs con el estado de React para capturar información del usuario.

# Instalacion 
- npm install
- npm run dev
